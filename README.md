# Smart Attendance System

This project is a simple and efficient attendance system developed using Python and OpenCV. It uses real-time face detection through a webcam to mark attendance automatically and store it in a CSV file.

## 🚀 Features
- Real-time face detection using OpenCV
- Automatically marks attendance with timestamp
- Stores data in a CSV file
- Simple and user-friendly implementation

## 🛠️ Technologies Used
- Python
- OpenCV
- CSV (for data storage)

## ▶️ How to Run
1. Install Python (3.x)
2. Install required library:
   pip install opencv-python
3. Run the program:
   python simple_attendance.py

## 📂 Output
- Attendance is saved in a file named `Attendance.csv`
- Format: Name, Time

## 🎯 Project Purpose
This project demonstrates the basic implementation of computer vision in real-world applications like attendance management systems.

## 🔮 Future Improvements
